## Install

[`Ayu Adaptive Personalized`](https://marketplace.visualstudio.com/items?itemName=ezh.ayu-adaptive-personalized)

## Development

Install dependencies

```shell
npm install
```

Update themes and Build VSIX package

```shell
npm run build && npm run package
```
